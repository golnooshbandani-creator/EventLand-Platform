## رویدادلند | EventLand

### درباره پروژه

**رویدادلند** یک پلتفرم مدیریت رویدادهای دانشگاهی است که با HTML، CSS، JavaScript و PHP ساخته شده است. کاربران می‌توانند رویدادها را مشاهده کرده، در آنها ثبت‌نام کنند و نظرسنجی پر کنند.

### امکانات
- 🌓 تم تاریک/روشن (ذخیره در مرورگر)
- ⏳ تایمر شمارش معکوس برای هر رویداد
- 🎠 اسلایدر تصاویر
- 📱 طراحی واکنش‌گرا (ریسپانسیو)
- 🔐 سیستم ورود و ثبت‌نام با PHP

### تکنولوژی‌ها

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

About The Project
EventLand is an academic event management platform built with HTML, CSS, JavaScript, and PHP. Users can view events, register for them, and fill out surveys.

Features
🌓 Dark/Light mode (saved in browser)
⏳ Countdown timer for each event
🎠 Image slider
📱 Responsive design
🔐 Login and registration system with PHP

Technologies
HTML5
CSS3
JavaScript
PHP
MySQL
